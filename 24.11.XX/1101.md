## CORS
### 동일 출처 정책(SOP) : 웹 보안 모델 중 하나, 웹 브라우저가 스크립트로 생성된 요청이 다른 출처로부터 리소스에 접근하는 것 방지
 - 웹 어플리케이션의 안정성 보호. CSRF, XSS 같은 웹 공격을 방지
 - 브라우저는 기본적으로 사용하는 정책
 - 출처 : 프로토콜, 호스트, 포트로 구성

### CORS
 - 교차 출처 리소스 공유(CORS) : 추가 HTTP 헤더를 사용.
 - 웹 어플리케이션은 리소스가 자신의 출처와 다르면...


#### CORS 해결
 - 서버 측 CORS 설정
 - 서버 측 프록시 설정
 - 클라이언트 측 설정
 - 프레임워크 & 라이브러리 지원
 - 일반적으로 서버 측에서 설정
